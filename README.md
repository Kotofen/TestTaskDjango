# Тестовое задание "Веб-сервис для управления движением денежных средств (ДДС)"

Данное веб-приложение, которое позволяет пользователям создавать, редактировать,
удалять и просматривать записи о движении денежных средств. Также оно предоставляет
интерфейс для работы со списками данных, необходимых для создания записей о ДДС.
Приложение реализовано на языке Python с использованием фреймворка Django.



## Инструкция к запуску проекта
1. Установка зависимостей проекта
```sh
$ pip install -r requirements.txt
```
2. Выполнение миграции для создания базы данных SQLite и базового заполнения списков
```sh
$ python TestTask\manage.py migrate
```

3. Локальный запуск приложения
```sh
$ python TestTask\manage.py runserver
```
